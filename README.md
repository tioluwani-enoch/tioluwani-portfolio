## Overview

This portfolio showcases work across:

- full-stack product systems
- automation and developer workflows
- machine learning pipelines
- embedded system interfaces
- production-minded debugging and iteration

The focus is not on decoration or isolated UI screens.

The focus is how systems are designed, shipped, and made useful under real constraints.

## Core Idea

Most useful software is not just a feature.

It is a set of responsibilities that need to stay clear:

- interfaces -> make workflows visible
- APIs -> move data predictably
- models -> classify, interpret, and support decisions
- systems -> execute reliably

This portfolio is structured around that separation.
